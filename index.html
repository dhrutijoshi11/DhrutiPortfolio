Tide Level Prediction

This project evaluates three GRU models for predicting tide time-series data at BHP. Starting with data preprocessing and splitting, each model—simple GRU (Model 1), varied layers and units (Model 2), and different activations (Model 3)—is trained and assessed. Metrics like MSE, RMSE, and MAE guide comparisons, alongside statistical tests and graphical analyses of predictions. Hyperparameter tuning optimizes each model, leading to the selection of the best-performing configuration for accurate tide predictions at BHP.

Languages and Tools: Python, pandas, Numpy, matplotlip, seaborn
Model Architecture: GRU(Gated Recurrent Unit)

Image Caption Generator

Employs Xception CNN for feature extraction and LSTM for caption generation from images.

×

Image Caption Generator

This project aims to develop an Image Caption Generator using Convolutional Neural Networks (CNNs) and Recurrent Neural Networks (RNNs) with Long Short Term Memory (LSTM) layers. Implemented in Python within a Jupyter notebook, the model utilizes TensorFlow and Keras. The dataset employed is Flickr 8K, comprising 8,000 images, each paired with five descriptive captions. The CNN extracts features and encodes input images, providing an image embedding as the initial input to the LSTM-based RNN, a distinctive feature of this architecture. Each step of the project, from data preprocessing to model training and evaluation, is meticulously documented to ensure clarity and reproducibility.

Languages and Tools: Python, pandas, Numpy, matplotlip, seaborn
Model Architecture: CNN And RNN with LSTM
